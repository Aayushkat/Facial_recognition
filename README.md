# Facial recognition with Vector databases
This a high speed facial recogniton and verifications system

### Key Technologies:
---
* **Python 3.11.0**: Main language
* **DeepFace**: Handles facial detection and vector encoding
* **Chromadb**: Store and search the facial embeddings
* **Tensorflow(2.20.0) and Keras(2.20.1)**:Core deep learning backend
